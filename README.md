# ChatVasp

基于 AutoGen 框架的多智能体协作系统，实现 VASP 计算的自动化工作流管理。

## 项目目标

通过多智能体协作，自动完成 VASP 第一性原理计算的全流程：
- **任务分解**：将复杂计算需求拆分为有序的子任务列表
- **输入文件生成**：自动生成 POSCAR、POTCAR、INCAR、KPOINTS 等输入文件
- **作业提交**：适配 HPC 集群环境，自动提交计算任务
- **结果分析**：分析计算结果并提供处理建议

## 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                   PLANNING_AGENT                        │
│              (任务规划与进度追踪)                          │
└─────────────────────┬───────────────────────────────────┘
                      │
        ┌─────────────┼─────────────┐
        ▼             ▼             ▼
┌───────────┐  ┌───────────┐  ┌───────────┐
│ VASP_FLOW │  │  SUBMIT   │  │  REVIEW   │
│   Agent   │  │   Agent   │  │   Agent   │
└─────┬─────┘  └───────────┘  └───────────┘
      │
      ▼
┌─────────────────────────────────────────┐
│            VASP Flow Agents             │
├─────────┬─────────┬─────────┬───────────┤
│  TASK   │  INCAR  │ KPOINTS │  POTCAR   │
│  Agent  │  Agent  │  Agent  │  Agent    │
└─────────┴─────────┴─────────┴───────────┘
```

## 快速开始

```bash
# 安装依赖
uv sync

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入 API 密钥

# 运行
python main.py
```

## 技术栈

- **AutoGen**: 多智能体框架
- **Pymatgen**: 材料科学计算库
- **MCP**: 文件系统操作
